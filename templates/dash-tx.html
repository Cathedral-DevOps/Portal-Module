Welcome to the Registrar! Developed by Cathedral Development https://cd-us.site. This software may look daunting, but do not be frightened! Here's a quick rundown of the environment. Sidebar Options;
'Registrar': View the active Registrations for the active event, or closest future event (considered active).
'Events': All upcoming events (add new events here!)
'Active Events': Select an event as active (auto-defaults to closest in time).
'Statistics' : View accumulated event retention in check-ins. (% attended over all events).
'Import To-Be' : Import registrations. Refer below.
